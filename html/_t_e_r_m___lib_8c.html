Detailed Description

TERM_Lib Source.

This library contains functions to allow for a simple terminal like communication with an embedded device. It can send character outputs over a arbritray channel, and provides a structure for command induced function calls

Converts a binary number into its decimal representation, and sends it out.

Converts a binary number into a length number of digits decimal representation. If the source value had more decimal digits, only the bottom length digits are given. After conversion the number is put onto the TERM channel.

Parameters
valbinary number to convert into a decimal output
lengththe number of decimals to show in the output

void TERM_PutHex ( uint8_t *  src,
uint8_t  nrBytes 
)

Converts a binary number into its hex representation, and sends it out.

Treats a number as an array of unsigned bytes, of which nrBytes LSBs are converted to HEX format, starting with the MSB of the LSBs. This is then sent out on the TERM channel

Parameters
srcpointer to the first byte to convert
nrBytesnumber of bytes to convert

Example Usage:

uint16_t value = 0x1234;
TERM_PutHex((uint8_t *)&value, sizeof(value)); //this gives 0x1234
TERM_PutHex((uint8_t *)&value, 1); //this gives 0x34
TERM_PutHex((uint8_t *)&value, 3); //this gives 0x??1234
