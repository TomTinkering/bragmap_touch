Detailed Description

STM32 RS232 driver hardware layer.

Hardware level calls required by the RS232 Generic driver RS232_drv.c This layer is mostly macro based although some more complex functions have an implementation here.

Set up the STM32 interrupt controller and USART interrupts.

This functions stores the higher level interrupt callback function pointers. Next the RX empty en TX empty interrupt are enabled which call the callback functions.
