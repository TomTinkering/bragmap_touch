Detailed Description

Generic RS232 driver, interrupt/buffer based.

Interrupt based RS232 implementation. This implementation uses a seperate RX and TX buffer, which are filles/emptied using interrupts. Relies on a lower level hardware implementation (hw_RS232_drv.h)

Configures the UART peripheral settings.

Sets the baudrate, number of databits, type of parity and the number of stopbits.

Note
In case one of the inputs is invalid, the function returns with a corresponding error. Some, or all of the other settings might be set depending on which setting failed.
Parameters
BaudRatebuad rate to be set in bits/s (2400,4800,9600.. up to 115200)
DataBitsThe number of bits per data packet (8 or 9)
ParityParity type used. N=none, E=even, O=odd;
StopBitsnumber of stopbits used (1 or 2)
Return values
RS232_INVALID_DATA_SIZEDatasize not supported
RS232_INVALID_PARITYGiven parity type not supported
RS232_INVALID_STOPBITSNumber of stopbits not supported
